# OrthoMCL

OrthoMCL是一种用于分组直系同源蛋白质序列的基因组规模算法。
它不仅提供了两个或多个物种/基因组共享的群体，而且提供了代表物种特异性基因扩增家族的群体。
因此它可作为自动化真核基因组的重要用途
